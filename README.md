## How to run
1. Clone this repository
2. Clone the [back-end repository](https://github.com/Nello-Moreira/Projeto13_MyWallet_back-end)
3. Follow instructions to run back-end
4. Install dependencies
```bash
npm i
```
5. Run the front-end with
```bash
npm start
```
